3. **Actionable Tips** - Specific, implementable suggestions
4. **Gentle Metrics** - Data presented as personal growth, not scores

## Implementation Guidelines

### UI Components
- Use calming colors (cyan, sky blue) as primary
- White cards on light backgrounds
- Smooth animations (no jarring transitions)
- Icons with purpose (never decorative alone)

### Copy Guidelines
- Short sentences (10-12 words max)
- Active voice
- No technical jargon
- Always encouraging tone

### Interactions
- Breathing effects on prompts
- Smooth state transitions
- Immediate visual feedback
- No time limits or countdowns
- Clear "you're done" signaling

### Error Handling
- Never blame the user
- Reframe as learning opportunity
- Suggest next steps gently
- Hide technical errors from view
